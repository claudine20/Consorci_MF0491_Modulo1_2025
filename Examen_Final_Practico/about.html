¿Qué es TaskPortal?


TaskPortal es una miniaplicación web que te permite gestionar tus tareas de forma sencilla y rápida. Puedes agregar nuevas tareas, marcarlas como completadas o eliminarlas. La aplicación también permite filtrar entre tareas completadas y pendientes.



Gracias a su diseño responsive, puedes usarla tanto en computadoras como en dispositivos móviles. Los datos se guardan localmente en tu navegador mediante localStorage, por lo que no se pierden al recargar la página.



Esta aplicación fue desarrollada como parte de un examen práctico, poniendo en práctica HTML, CSS y JavaScript, incluyendo validaciones, manipulación del DOM, y buenas prácticas de desarrollo web.
